# Blog Pipeline - SEO-gesteuerte Content-Automatisierung

**Stand:** 2026-01-28
**Status:** In Planung

---

## Übersicht

Automatisierte Pipeline für Blogartikel basierend auf AHREFS Keyword-Erkenntnissen.

## Workflow (geplant)

```
AHREFS Keywords → Themen-Auswahl → Content-Erstellung → Review → Veröffentlichung
```

### 1. Keyword-Recherche (AHREFS)
- [ ] AHREFS-Export-Format definieren
- [ ] Relevante Metriken: Suchvolumen, KD, CPC, Traffic Potential
- [ ] Filter-Kriterien für neurealis-relevante Keywords

### 2. Themen-Priorisierung
- [ ] Scoring-Modell für Keyword-Auswahl
- [ ] Cluster-Bildung (Pillar + Supporting Content)
- [ ] Redaktionsplan-Integration

### 3. Content-Erstellung
- [ ] Briefing-Template
- [ ] KI-gestützte Entwürfe (optional)
- [ ] SEO-Checkliste pro Artikel

### 4. Review & Optimierung
- [ ] Qualitätskriterien
- [ ] SEO-Check vor Veröffentlichung

### 5. Veröffentlichung & Tracking
- [ ] Publishing-Workflow
- [ ] Performance-Tracking (GSC, Analytics)
